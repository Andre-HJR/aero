RoundRobin

Round Robin is the simplest algorithm for a preemptive scheduler. When the system timer fires, the next process in the queue is switched to, and the preempted process is put back into the queue.

The kernel idle process is a special kind of process that is run when no processes in the scheduler’s queue are avaliable to execute. The idle process is to be created for each CPU.
