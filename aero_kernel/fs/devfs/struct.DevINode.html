Struct aero_kernel::fs::devfs::DevINode

source · 
pub struct DevINode(Arc<dyn Device>);
Expand description

Structure representing a device inode. This is internally used by ram-fs to create a new inode with the file type of device and its contents as a reference-counting pointer to the device itself.
